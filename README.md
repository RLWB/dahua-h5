#### 开发注意

- 图片资源存放在assets目录下，按照页面区分文件夹

- 命名用英文，手动去翻译一下

- 图片命名规则 dahua_[页面名称]_[图片名称].***

- 公用组件swiper，放在main.js里面全局注册

- class 命名规范 [页面名称]-[-[...]]

- 组件样式统一scss语法，设置scoped样式隔离

- 写页面的时候，如果觉得像素偏差比较大，以psd为准

- 静态图片使用之前https://tinypng.com/压缩一下

- 一切参考这个
